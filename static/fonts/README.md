# Fonts

`static/fonts/excalidraw/generated/` contains the self-hosted font subsets used by generated Excalidraw SVGs.

Those files are generated by [tools/excalidraw-exporter/main.js](/Users/dmytro/work/github/blog-dmytro/tools/excalidraw-exporter/main.js) from Excalidraw's SVG font export output, then written as content-addressed WOFF2 files so SVGs can reference local assets instead of embedding `data:` fonts.
